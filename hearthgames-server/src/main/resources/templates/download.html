HEARTHGAMES.com

Official Hearth Games Client Uploader

The Uploader is a Java program that runs on your computer. It reads the Hearthstone log file as it is being written and uploads games as they finish. Currently only Casual and Ranked Play mode games are tracked. In the future Arena and Tavern Brawl will be supported.

If you don't have the Java Runtime you can download it here.
