ENABLE LOGGING

First thing you have to do is enable logging before you can find any log files. Download log.config and put it in the directory specified below:

  • Windows
  • Mac
  • Android
C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\
~/Library/Preferences/Blizzard/Hearthstone/
Android/data/com.blizzard.wtcg.hearthstone/files

WHERE DO I FIND MY LOG FILES?

  • Windows
  • Mac
  • Android
  • Timestamps / 5MB file size limit: C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\Logs\hearthstone_2015_11_24_12_53_07.log
  • No Timestamps / No file size limit: C:\Program Files (x86)\Hearthstone\Hearthstone_Data\output_log.txt
  • Timestamps / 5MB file size limit: ~/Library/Preferences/Blizzard/Hearthstone/Logs/hearthstone_2015_11_24_12_53_07.log
  • No Timestamps / No file size limit: ~/Library/Logs/Unity/Player.log
  • Timestamps / 5MB file size limit: Android/data/com.blizzard.wtcg.hearthstone/files/Logs/hearthstone_2015_11_24_12_53_07.log
  • No Timestamps : Currently not available.

Warning

Timestamped log files are limited in size by Hearthstone to 5MB, which is equivalent to about 2 games worth of data. After 5MB logging is truncated. You will have to restart Hearthstone to create a new log file.
