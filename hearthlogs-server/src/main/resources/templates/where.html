HEARTHLOGS.com A Hearthstone Log Analyzer

WHERE DO I FIND MY LOG FILES?

Hearthstone Log File Locations:

C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\Logs

~/Library/Preferences/Blizzard/Hearthstone/Logs

Android/data/com.blizzard.wtcg.hearthstone/files/Logs

Log Config File:

By default Hearthstone does not enable it's many loggers. It simply logs System Information. To allow Hearthlogs.com to analyze your log files you will need to create a file called log.config.

C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\log.config

~/Library/Preferences/Blizzard/Hearthstone/log.config

Android/data/com.blizzard.wtcg.hearthstone/files/log.config

In the contents of that file place the below lines. Please note that you will need to exit Hearthstone and restart it for the logging to happen.

[Power]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false
[Asset]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false
[Bob]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false

Android Instructions

Get an App like File Manager (File Transfer) by Cheetah Mobile and follow these instructions:

  • Open the app, go into "Device"
  • Navigate to Android/data/com.blizzard.wtcg.hearthstone/files and create the log.config file by click the top right corner and choosing "Create"
  • With the log.config created edit it with the Text Editor and enter the lines as above
  • After you have played a game you can select the log file and click "Share" and choose a method of transferring it off your device

Warning

Hearthstone keeps only 5 log files at a time, each time you start Hearthstone it will create a new log file, and if there were more than 5 it will delete the oldest one. Make sure you save them to a different location if you want to keep all games played.
