WHERE DO I FIND MY LOG FILES?

Before you can actually get any log files you need to enable logging. See below for the Windows/Mac/Android instructions of where specifically to find the log.config location.

You will need to create the log.config file with the contents below. Please note that you will need to exit Hearthstone and restart it for the logging to happen.

[Power]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false
[Asset]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false
[Bob]
LogLevel=1
FilePrinting=false
ConsolePrinting=true
ScreenPrinting=false

There are 2 kinds of log files. One that does not contain timestamped log entries and one that does. The first log file output_log.txt (Windows) / Player.log (Mac) does not contain timestamps. On the other hand there is the hearthstone_2015_11_24_12_53_07.log file that contains timestamps. This will give you more detailed information about your game such as how long that game took and the duration of each turn.

Warning

If you choose to upload log files that have timestamps be warned that Hearthstone keeps only 5 log files at a time in that directory and the file size is limited to 5MB. Unfortunately you can probably only get 2-3 games recorded in there before Hearthstone stops logging and just truncates the file. If you exit Hearthstone it will start a new log file, keeping a maximum of 5 around. If you want this kind of log file you will need to exit hearthstone every couple games and copy the file.

The other log file does not seem to have a file size limitation that I know of. This is the log file most of the other deck/log trackers use.

Windows Log File Locations:

No timestamps / No file size limit: C:\Program Files (x86)\Hearthstone\Hearthstone_Data\output_log.txt

Timestamps / 5MB file size limit: C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\Logs\hearthstone_2015_11_24_12_53_07.log

Windows log.config Location:

C:\Users\yourusername\AppData\Local\Blizzard\Hearthstone\

Mac Log File Locations:

No timestamps / No file size limit: ~/Library/Logs/Unity/Player.log

Timestamps / 5MB file size limit: ~/Library/Preferences/Blizzard/Hearthstone/Logs/hearthstone_2015_11_24_12_53_07.log

Mac log.config Location:

~/Library/Preferences/Blizzard/Hearthstone/

Android Instructions

log.config location: Android/data/com.blizzard.wtcg.hearthstone/files

Get an App like File Manager (File Transfer) by Cheetah Mobile and follow these instructions:

  • Open the app, go into "Device"
  • Navigate to Android/data/com.blizzard.wtcg.hearthstone/files and create the log.config file by click the top right corner and choosing "Create"
  • With the log.config created edit it with the Text Editor and enter the lines as above
  • After you have played a game you can select the log file and click "Share" and choose a method of transferring it off your device
